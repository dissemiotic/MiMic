YouTube Channel Handle Settings

Approved Channels

Enter YouTube channel handles below, one per line. Each handle must start with and contain only one instance of /@.
The default list consists of YT horror reviewers, commentators, etc., that the developer found during their own scouring of the YT scene. Is the default list missing your favorite commentator? Send us a note at dissemiotic@gmail.com!

How to find a channel handle

  1. Visit the YouTuber's page
  2. Look at the page's URL - it should be in the format youtube.com/@channelname
  3. Copy the handle portion of the URL, including the /@ prefix
  4. Paste it into the list above and save!
